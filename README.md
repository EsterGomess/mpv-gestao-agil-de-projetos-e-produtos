# 📚 Cadernito

> MVP developed for the lecture **Agile Project and Product Management** course of post degree Software Engineering at the Puc - RIO.

## 📖 About the Project

**Cadernito** is a web application designed for students who want to improve knowledge retention using the **Spaced Repetition** learning methodology.

The system allows users to create study topics, manage flashcards, and automatically schedule review sessions at the most effective times to reinforce long-term memory.

This MVP was designed following the practices of **Lean Inception**, **Scrum**, **Requirements Engineering**, and **User Interface Design**, based on the concepts learned throughout the course.

---

# 🎯 MVP Goal

Validate whether university students are willing to use a simple web platform to organize their studies through flashcards and spaced repetition.
